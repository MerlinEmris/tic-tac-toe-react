# Introduction

This is a [Next.js](https://nextjs.org/) app inspired from tic-tac-toe game!

## Getting Started

First install packages:

```bash
yarn install
```


run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
